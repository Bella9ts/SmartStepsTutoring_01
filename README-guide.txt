SMARTSTEPS TUTORING WEBSITE GUIDE

WHAT IS INCLUDED
- index.html = homepage
- about.html = about page
- services.html = services page
- testimonials.html = testimonials page
- contact.html = contact page with sample form and Calendly embed
- style.css = colours and layout styling
- assets/logo.png = logo image taken from your flyer

BEFORE YOU PUBLISH
1. Open each HTML file in a browser and read the text.
2. Replace the sample tutor background with your real information.
3. Update prices, lesson times, and service details.
4. Replace the sample testimonials with real ones when you have them.
5. On contact.html, replace the sample Calendly link with your real Calendly booking URL.
6. If you want the contact form to actually send messages, connect it to a service such as:
   - Formspree
   - Netlify Forms
   - Basin

PLAIN ENGLISH: DOMAIN AND HOSTING
A domain name is your website address, for example:
www.smartstepstutoring.co.uk

Hosting is the online service that stores your website files and makes them visible on the internet.

GOOD BEGINNER OPTIONS FOR A STATIC WEBSITE
1. GitHub Pages
   Best if you do not mind a few technical steps.
   Very good for free hosting.

2. Netlify
   Best for beginners.
   Easy drag-and-drop deploy.
   Also good if you want forms later.

3. Cloudflare Pages
   Also very good and fast.
   Slightly more technical than Netlify, but still beginner-friendly.

SIMPLE RECOMMENDATION
For a complete beginner, Netlify is usually the easiest.
Buy your domain from a domain registrar, then connect it to Netlify.

HOW TO BUY A DOMAIN NAME
You can buy a domain from:
- Namecheap
- GoDaddy
- 123 Reg
- Google Domains alternatives through Squarespace Domains or other registrars

Choose a domain name that is:
- short
- easy to spell
- close to your business name
- preferably .co.uk or .com

Example ideas:
- smartstepstutoring.co.uk
- smartstepslearning.co.uk
- smartstepstutor.co.uk

OPTION A: FASTEST WAY TO GO LIVE WITH NETLIFY
1. Create a free Netlify account.
2. Put all website files into one folder.
3. Drag that folder into Netlify's deploy area.
4. Netlify will publish the site with a temporary web address.
5. Test the site.
6. In Netlify settings, choose "Add custom domain".
7. Enter the domain name you bought.
8. Netlify will show DNS records to add at your domain registrar.
9. Copy those DNS records into your domain account.
10. Wait for the connection to finish. This can take a few minutes or up to 24 hours.
11. Your site will then open with your custom domain.

OPTION B: GITHUB PAGES
1. Create a free GitHub account.
2. Create a new repository.
3. Upload all website files.
4. Go to Settings > Pages.
5. Choose the main branch as the source.
6. Save.
7. GitHub will publish your site with a github.io address.
8. Later, connect your custom domain in the Pages settings.
9. Add the DNS records shown by GitHub at your domain registrar.

OPTION C: CLOUDFLARE PAGES
1. Create a Cloudflare account.
2. Create a new Pages project.
3. Upload your website files or connect GitHub.
4. Publish the site.
5. Add your custom domain inside Cloudflare.
6. Follow the DNS instructions it gives you.

HOW TO VIEW THE WEBSITE ON YOUR COMPUTER FIRST
1. Open the folder.
2. Double-click index.html.
3. The homepage will open in your browser.
4. Click the menu links to test the other pages.

HOW TO EDIT THE WEBSITE

HOW TO MAKE THE CONTACT FORM WORK
A plain HTML file does not send messages by itself.
The easiest beginner options are:
- Netlify Forms if you host on Netlify
- Formspree if you want a simple external form service
